# SEO Style Guide

- Language: Spanish (es-MX), beginner friendly.
- Timeframe: write as if it is early 2026.
- Keep tone helpful, non-clickbait, AdSense safe.
- Include Mexico context: MXN, megas, fibra, datos moviles.
- Meta description must be <= 160 characters.
- Use clear H2/H3 structure, table of contents, FAQ, CTA.
- Add internal links to pillar hubs, siblings, and related pillars.
